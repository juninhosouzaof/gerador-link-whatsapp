VEM DE ZAP CONTATO

Sobre o projeto VEMDEZAP

O projeto tem como objetivo permitir que o usuário digite o número a ser contatado e também insira a mensagem a ser enviada para esse contato, sem que necessite adicionar o número em sua lista de contatos pessoal. São apresentadas duas opções: "Gerar link" e "Abrir conversa já com o texto digitado".


Opção "Gerar link"

Permite que o usuário apenas gere o link, e este pode ser gerado com ou sem alguma mensagem pré-definida pelo usuário. Essa função é bem útil para profissionais autônomos que desejam inserir um contato rápido através do WhatsApp em seus perfis de redes sociais por exemplo.

Opção "Abrir conversa com a mensagem digitada"

Nesta opção é possível abrir a conversa diretamente já com a mensagem pré-digitada e pronta para ser enviada para o número desejado, porém, o usuário pode conferir a mensagem antes de efetivar

Tecnologias utilizadas

O projeto foi construído utilizando HTML5, CSS3, Javascript. O logotipo foi desenvolvido de maneira autoral através do Adobe Illustrator.
